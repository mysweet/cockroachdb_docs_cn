# Go 实现

选择语言很重要。速度、稳定性、可维护性：所使用语言的每一个特性能影响 CockroachDB 发展多块以及它工作多好。不是所有的语言都是生来平等的。Go 是一门开源的编程语言，作为 C++ 和 Java 的可行替代品，主要在谷歌开发。

-   构建分布式系统的优秀环境
-   编译为原生性能
-   垃圾收集和类型安全提供了稳定性
-   可读、文档良好的代码鼓励开源贡献
